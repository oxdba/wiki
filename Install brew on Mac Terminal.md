# Install brew on Mac Terminal
1. Open the terminal
2. Install xcode: ```xcode-select --install```
3. Install brew:
```
curl -O https://raw.githubusercontent.com/Homebrew/install/master/install.sh
more install.sh
chmod +x install.sh
./install.sh
```
